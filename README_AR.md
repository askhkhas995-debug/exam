# Arabic README

Arabic documentation is not maintained in this local preview. Use `README.md`,
`README_EN.md`, and `docs/STUDENT_USAGE.md` for the current delivery notes and
command references. Current terminology is documented there: Piscine uses
Moulinette through `pforge moulinette`, module summaries use
`pforge moulinette summary`, Exam uses Grademe through `pforge grademe`,
`pforge correct` dispatches to the active correction mode, and Vogsphere is a
local educational project workflow only.
