# Internal Artifacts

This directory contains handoff and reconstruction notes used while rebuilding
PiscineForge. These files are useful for maintainers, but they are not part of
the student-facing product surface and should be excluded from clean public
release packages unless intentionally published as project history.

The active package version is defined in `pyproject.toml`,
`piscine_forge/__init__.py`, and root `VERSION.txt`.
