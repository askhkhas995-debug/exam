# سياسة Norminette

Norminette هي طبقة إلزامية لمعظم مشاريع C في المسار الطبيعي والامتحانات.

## السياسة

- لا نعيد بناء Norminette من الصفر.
- نشغل `norminette` كأمر خارجي.
- نضيف pre-checks للسرعة والوضوح.

## pre-checks

```txt
- أسماء الملفات
- أسماء الدوال
- no forbidden functions
- no extra files
- no main in function exercises
- compile flags
```
