Write the following function:

```c
void print_nth_char(char *str, int n);
```

The function prints every nth character of `str`, followed by a newline. Counting is 1-based. If `n <= 0`, if `str` is empty, or if `n` is greater than the string length, print only a newline.

Examples: `ABCDEF, 2 -> BDF`; `Hello, 2 -> el`.

Allowed functions: `write`.
